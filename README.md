# Magnitude_Comparator

A 4-Bit Magnitude Comparator designed and implemented using Verilog HDL, digital logic ICs, and FPGA development tools.

This project compares two 4-bit binary numbers and determines whether:

  A > B
  
  A < B
  
  A = B
  

**PROJECT OVERVIEW**

A magnitude comparator is a combinational circuit used to compare binary numbers.
This project includes:

  Logic design of a 4-bit comparator
  
  Verilog HDL implementation
  
  Testbench simulation
  
  FPGA implementation using the Basys 3 board
  
  Hardware implementation using logic gate ICs
  

**IMPLEMENTED USING**

  Verilog HDL
  
  Vivado Design Suite
  
  Basys 3 FPGA Board
  
  Digital Logic ICs
  

**LOGIC GATE ICs USED**

  IC 7408 => 2-input AND Gate
  
  IC 7402 => 2-input NOR Gate
  
  IC 7411 => 3-input AND Gate
  
  IC 7421	=> 4-input AND Gate
  
  IC 7404 => NOT Gate
  
  IC 4072 => 4-input OR Gate

  
**TOOLS AND TECHNOLOGIES**

  Verilog HDL
  
  Vivado Design Suite
  
  Basys 3 FPGA Board
  
  Digital Electronics Hardware Components
